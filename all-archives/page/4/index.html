计算机网络

计算机网络定义：已能够相互共享资源的方式互联起来的自治计算机系统的集合。
特征：共享软件、硬件以及数据等资源；分布在不同地理位置的“自治系统”；遵循一定的规则——协议

计算机网络和分布式系统区别：计算机网络目的是共享资源，对外呈现分散，分布式系统目的是以提高整体性能，对外呈现整体。

分层的优点：各层独立、灵活性好、结构上分开、促进标准化

数据链路层

数据链路层协议提供的最基本的功能：数据在数据链路上的正常传输、帧定界（帧同步）、差错控制、流量控制、寻址

向网络层提供的三种基本服务：无确认无连接服务、有确认无连接服务、面向连接确认服务

应答方式：正向应答、负向应答、双向应答

成帧：字符计数法、带字符填充的首尾字符定界法、带位填充的首尾标记定界法（+01111110，五个1后加0）、物理层编码违例法

差错控制
随机、连续突发
帧出错、帧丢失
方法：自动纠错机制（FEC）,检错反馈重发机制（ARQ）
码字、纠错码、检错码

编码的海明距离
海明定理
检测d位错误，编码海明距离至少为d+1
纠正 2d+1

差错码举例：奇偶校验码、垂直水平奇偶校验码、交织奇偶校验码
循环冗余码CRC
检验和算法

单工停等协议
连续ARQ协议
滑动窗口协议：控制已发送但未确认的数据帧的个数（滑动窗口的大小）
接受窗口

SLIP协议
PPP协议（包括链路控制协议LCP、网络控制协议NCP）

局域网

局域网是一种将小区域内的各种通信设备互联在一起的通信网络

IEEE 802局域网：数据链路层（逻辑链路控制子层、介质访问控制子层）、物理层

物理层功能：信号编码\译码、前导码的生成\去除、比特的发送\接收

数据链路层的功能：

MAC子层：成帧\拆帧，实现、维护MAC协议，位差错检测，寻址

LLC子层：向高层提供SAP，建立/释放逻辑连接，差错控制，帧序号处理，提供某些网络层功能

信道共享技术
关键问题：如何解决对信道争用

解决信道争用的协议：介质访问控制协议MAC，是数据链路层协议的一部分

静态分配：频分多路复用FDM（波分复用WDM）、时分多路复用TDM
动态分配：冲突协议、无冲突协议

冲突协议

ALOHA协议：纯ALOHA协议和时间片ALOHA协议

载波监听多路访问协议CSMA：1-坚持性CSMA、非坚持型CSMA（有冲突时是否继续监听\等待随机时间）、p-坚持性CSMA（p的概率发送，1-p的概率延迟）

带冲突检测的载波监听多路协议CSMA/CD（有冲突就发送干扰信号并退避）

令牌协议：有令牌才可发送

强弱、动静、显隐类型的理解

12 1月 2018

类型

强类型、弱类型

  • 强类型strongly typed: 如果一种语言的所有程序都是well behaved——即不可能出现forbidden behaviors，则该语言为strongly typed。

  • 弱类型weakly typed: 否则为weakly typed。比如C语言的缓冲区溢出，属于trapped errors，即属于forbidden behaviors..故C是弱类型前面的人也说了，弱类型语言，类型检查更不严格，如偏向于容忍隐式类型转换。譬如说C语言的int可以变成double。这样的结果是：容易产生forbidden behaviours，所以是弱类型的

动态、静态类型

  • 静态类型 statically: 如果在编译时拒绝ill behaved程序，则是statically typed;

  • 动态类型dynamiclly: 如果在运行时拒绝ill behaviors, 则是dynamiclly typed

显式与隐式

显式定义类型和隐式定义类型

弱类型、动态类型检查： Perl/PHP

签到题，贪心，总是取当前数目最大的三种，各种解法。

#include<iostream>
#include<cstdio>
#include<bits/stdc++.h>
#define mp make_pair
#define pb push_back
#define lowbit(i) (i&(-i))
#define LL long long
using namespace std;
struct point{
    int num,cnt;
    bool operator < (const point & a)const {
        if(cnt == a.cnt)return num < a.num; 
        return cnt < a.cnt;
    }
}; 
priority_queue<point>q;
vector<int >v;
int main(){
    std::ios::sync_with_stdio(false);
    int T;
    cin>>T;
    while(T--){
        int n,term;
        v.clear(); 
        while(q.size() >0)q.pop();
        cin>>n;
        for(int i = 1; i <= n ; ++i){
            cin>>term;
            v.pb(term);
        }
        sort(v.begin(), v.end());
        int temp = -1; point nn;
        for(auto i : v){
            if(temp == -1){
                temp = i;
                nn.num = i; nn.cnt = 1;
            }else if(temp != i){
                q.push(nn);
                nn.num = i; temp = i;
                nn.cnt = 1;
            }else ++nn.cnt; 
        }
        q.push(nn);
        LL ans = 0;
        while(q.size() >= 3){
            point a = q.top(); q.pop();
            point b = q.top(); q.pop();
            point c = q.top(); q.pop();
        //    cout<<a.num<<" "<<b.num <<" "<<c.num<<endl;
//每次只能取一个，不然不满足贪心正确性
//            a.cnt -= c.cnt;
//            b.cnt -= c.cnt;
//            ans += c.cnt;
            --a.cnt; --b.cnt;--c.cnt;
            ++ans;
            if(a.cnt > 0)q.push(a);
            if(b.cnt > 0)q.push(b);
            if(c.cnt > 0)q.push(c); 
        }
        cout<<ans<<endl;

    }

    return 0;
}

Plates

数学题，简单思维。
只要把n个圆的圆心围成正多边形,然后求出这个正多边形的外切圆半径再加上小圆的半径就解决了。记得特判一下n<=2的情况。

### AC代码 ###
#include<iostream>
#include<cstdio>
#include<bits/stdc++.h>
#define mp make_pair
#define pb push_back
#define lowbit(i) (i&(-i))
#define LL long long
using namespace std;
const double pi = acos(-1.0);

int main(){
    std::ios::sync_with_stdio(false);
    int n,r,R;
    while(cin>>n>>R>>r){
        if(n <= 2)cout<<(R >= r * n ? "yes":"no") <<endl;
        else{
            double du = (n-2)*1.0/n * pi;
            double m = r + r / cos(du/2);
            //cout<<m<<endl;
            if(m > R)cout<<"no"<<endl;
            else cout<<"yes"<<endl;    
        }

    } 
    return 0;
}

博弈论-k倍取数

24 8月 2017

斐波那契取数

   训练时遇到一道博弈题，经典博弈模型-k倍取数。直接入手比较难理解，于是去找了下 “k = 2” 的斐波那契取数模型先了解一波。

原理引入

(1):齐肯多夫定理： 任意数可分解为斐波那契数列中数的和
(2)当石头总数n为斐波那契数的时候，后手必赢。

证明2参考

好像又说绕回来了。。。有点迷

k倍动态减法游戏

  原理详述 这篇博客讲的非常清晰。推荐。
将k倍动态减数问题和斐波那契数列结合起来，仿照斐波那契数列来构造a数组。

    即 a[ i + 1 ] = a[ i ] + sgma( j = 1,a[j] k < a[i]>) ( a[ j ] + 1);
            其中a[ j ] k < a[ i ]。
  引入b数组表示a[ i ] + a[ j ], 表示a数组前i项所能构造的最大数.

  简化为

     a[ i + 1] = a[ i ] + b[ j ];
    其中代表所有满足 a[j] * k < a[i] 的a[ j ]和

  不过之后的确搞通了，感谢前行者们无私的分享。
