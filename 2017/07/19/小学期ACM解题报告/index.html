#拖了这么久的解题报告#

##Day1##

##A题##
  是个简单的贪心题，如果题目要求改成要求是不同的素数的和要就稍微需要考虑一点东西了，很遗憾，一开始就题目读错了，水题反而wa了。
  因为是要拆分成最多个数的素数和，所以以2和3去拆就好，做到O（1）的并不难。

B题

这是一个纯粹新手题，不管是暴力枚举还是排序后二分查找思路都是很清晰的。

C题

思路很好想，稍微写下几次操作之后就会发现6次一个循环，所以剩下的思路就是把opt次数模6然后剩下无论是暴力还是打表都是很轻松的。

D 题

  Day1的题毕竟是为了营造友好氛围的，没什么难度。写题解不过是为了完整性。
  这道题其实就是权重平衡之后最少的那个。很粗暴的计算就好。

ans = min(a,b/2,c/4)*7。

E 题

其实还是很有难度的，但对于时间不是很限制，那就是一个抖机灵的题。因为是按行按列翻转，所以很容易想到每行每列最多翻转一次。所以从行上考虑，只要保证不符合最终状态的都翻转1次，最后统计所有列是否是符合的就好。

F 题

这是一道裸的矩阵快速幂。

f(x) = a * f(x - 1)  + b * f(x - 2) + c * f(x - 3)

f(1) = f(2) = f(3) = t;

[f(x+1),f(x),f(x-1)]T = [a,b,c][f(x),f(x-1),f(x-2)]T
