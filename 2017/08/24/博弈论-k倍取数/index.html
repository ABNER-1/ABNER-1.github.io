博弈论-k倍取数

24 8月 2017

斐波那契取数

   训练时遇到一道博弈题，经典博弈模型-k倍取数。直接入手比较难理解，于是去找了下 “k = 2” 的斐波那契取数模型先了解一波。

原理引入

(1):齐肯多夫定理： 任意数可分解为斐波那契数列中数的和
(2)当石头总数n为斐波那契数的时候，后手必赢。

证明2参考

好像又说绕回来了。。。有点迷

k倍动态减法游戏

  原理详述 这篇博客讲的非常清晰。推荐。
将k倍动态减数问题和斐波那契数列结合起来，仿照斐波那契数列来构造a数组。

    即 a[ i + 1 ] = a[ i ] + sgma( j = 1,a[j] k < a[i]>) ( a[ j ] + 1);
            其中a[ j ] k < a[ i ]。
  引入b数组表示a[ i ] + a[ j ], 表示a数组前i项所能构造的最大数.

  简化为

     a[ i + 1] = a[ i ] + b[ j ];
    其中代表所有满足 a[j] * k < a[i] 的a[ j ]和

  不过之后的确搞通了，感谢前行者们无私的分享。

int main(){
    int T,n,k,num = 0;
    cin >> T;
    while(++num <= T){
        cin>>n>>k;
        a[0] = b[0] = 1;
//        有误

//            if(j < 0)b[i] = a[i];
//            else b[i] = a[i] + b[j];
//        } 
        int i = 0, j = 0;  
        while(a[i]) < n{         // 构建数列  
            ++i;
            a[i] = b[i - 1] + 1;  
            while(a[j + 1] * k < a[i])++j;  
            if(k * a[j] < a[i])  
                b[i] = b[j] + a[i];  
            else b[i] = a[i];  
        }  
        printf("Case %d: ",num);
        if(a[i] == n)printf("lose\n");
        else {
            int ans;
            while(n){
                if(n >= a[i])n -= a[i],ans = a[i];
                --i;
            }
            printf("%d\n",ans);
        }
    } 
    return 0;
}
