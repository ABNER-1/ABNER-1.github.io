eNSP学习笔记

20 4月 2018

组网练习命令记录

VLAN操作

sys 进入系统模式

新建vlan

分配vlan到端口：

  • 第一种

    修改端口类型 改为access : port link-type access
    分配端口vlan : port default vlan 100

  • 第二种

    vlan 100
    port ethernet 0/1

进入vlanif接口（三层）分配ip

ip address 10.1.1.1 24

OSPF

  • 新建loopback口 

    loopback0

  • 配置router id

    router id + ip地址（loopback地址）

  • 新建ospf进程

    ospf + 进程号

  • 创建area

    area + area号

  • network宣告 

    network + 路由 + 反掩码

  • 打印ospf信息

    display ospf peer

VLAN

vlan 隔离了二层广播域

vlan头部

目的地址 源地址 tag信息 + 报文

基于端口的vlan

端口类型 Access 、 Trunk 、 Hybrid端口

Access类型端口只属于一个VLAN，有pvid（多面向主机）

Trunk允许多个VLAN通过 （多面向交换机、路由器）

VLAN间通信方式:单臂路由、三层交换机

单臂路由

交换机上配置trunk端口：

进入对应端口
port link-type trunk
port trunk allow-pass vlan all

路由器上：

interface ethernet 0/1.1   建立子接口
control-vid 100 dot1q-termination   
控制vlan  编号100  协议

dot1q termination vid 100
                  vlan编号
arp broadcast enable
ip address 192.168.10.1 255.255.255.0  分配网关

端口聚合

用于交换机之间，增加带宽、冗余的好处等

创建eth-trunk

interface Eth-Trunk1

进入interface 0/0/2并将其加入trunk1

interface G 0/0/2    
eth-trunk 1      加入trunk1

interface G 0/0/3
eth-trunk 1

进入eth-trunk1接口进行配置

interface eth-trunk 1  
port link-type trunk   端口类型为trunk
port trunk allow-pass vlan 30
     配置trunk允许vlan30 通过

静态路由

配置静态路由
ip route-static 192.168.1.10 24 10.1.1.2
                目的ip地址  子网掩码    下一跳地址
或
ip route-static 192.168.1.10 24 Ethernet 0/0/0
                                下一跳出口
打印路由表信息
display ip routing-table
