参数含义

参数名 参数解释
url 请求的链接地址
params 请求的参数及对应值
headers 请求headers的参数及对应值
charset 字符集
HttpClient 请求的Client，可选项对应为Http和Https
HttpConfig 可配置所有config的类，最终由execute执行后返回HttpRespone对象
HttpContext HttpContext表示HTTP进程的执行状态。它是一种可用于将属性名称映射到属性值的结构。
headers(Headers) 请求的headers的参数及对应值,用apache.http.Header进行封装
cookieStore CookieStore类对象,Cookies的抽象存储

execute()方法 对 request、charset进行处理后调用HttpClient.execute方法执行请求操作

try {
    res = HttpClientUtil.doPost(MATRIX_URL, map);
} catch (Exception e) {
    e.printStackTrace();
}
