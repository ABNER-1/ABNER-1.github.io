git reverse

  • git reset
    • git reset –hard
      彻底回到 的那一次 commit，这中间发生的变更会完全消失（可以通过组合 git cherry-pick 来补充回来）
    • git reset –soft
      回到 的那一次 commit，这中间发生的变更不会消失，只是撤销中间的commit
  • git revert # 提交一个新的commit去revert
